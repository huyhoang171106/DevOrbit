Dark Space / Cosmic — Which accent?

The base will be deep navy (#0a0e1a) with glass-morphism cards. The accent color defines the personality.

A

Cyan / Teal

Like a spacecraft control panel. Cold, technical, precise. Pairs well with the "orbit" concept — think orbital tracking displays.

B

Amber / Orange

Warm accent against deep space. Like the sun or a distant star. High contrast, feels alive and energetic.

C

Emerald / Mint

The current brand-green but adjusted for dark space. Keeps some continuity. Feels like a green nebula.

D

Violet / Purple

Cosmic nebula vibes. More creative/artistic than technical. Pairs with deep indigo backgrounds.
