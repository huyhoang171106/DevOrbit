# Curriculum: Software Engineering (Class 2025)

**Source**: [DAA UIT](https://daa.uit.edu.vn/content/cu-nhan-nganh-ky-thuat-phan-mem-ap-dung-tu-khoa-20-2025)  
**Status**: Active (from 2025)  
**Credits**: 130  
**Duration**: 4 Years (8 Semesters)

## Learning Outcomes (LO)

| LO | Description |
| --- | --- |
| **LO 1** | Foundation in natural and social sciences. |
| **LO 2** | Knowledge in IT and Software Engineering. |
| **LO 3** | Analysis and proposal of SE solutions; lifelong learning. |
| **LO 4** | System design, implementation, and evaluation. |
| **LO 5** | Communication and teamwork. |
| **LO 6** | English proficiency for technical communication. |
| **LO 7** | Leadership and management skills. |
| **LO 8** | Professional ethics and legal responsibility. |

## Course Structure

### 1. General Education (43 Credits)
- **Politics/Law (13 TC)**: SS003, SS007, SS008, SS009, SS010, SS006.
- **Math/Science (18 TC)**: MA006 (Calculus), MA003 (Linear Algebra), MA004 (Discrete Structures), MA005 (Probability & Statistics), IT001 (Intro to Programming).
- **English (12 TC)**: ENG01, ENG02, ENG03.

### 2. Core Professional (49 Credits)
- **Compulsory (37 TC)**:
    - SE005: Intro to SE
    - IT002: OOP
    - IT003: Data Structures & Algorithms
    - IT004: Database
    - IT005: Intro to Networking
    - IT007: Operating Systems
    - IT012: Computer Architecture II
    - IT008: Visual Programming
    - SE100: OOP Software Development Methods
    - SE104: Intro to Software Engineering
- **Elective (12 TC)**: DevOps (SE359), Game Dev (SE102/115/116), Mobile (SE114), Cloud (SE360), Microservices (SE361), OSS (SE301), HCI (SE215), Testing (SE113), PM (SE358).

### 3. Specialized Electives (16 Credits)
- Java (SE330), Advanced DB (SE332), Web Tech (SE347), .NET (SE310), Mobile App (SE346), VR (SE352).
- Software Architecture (SE356), Requirement Analysis (SE357), J2EE (SE325).
- Game Design (SE220), 3D Graphics (SE320), Game AI (SE328), etc.

### 4. Graduation (16 Credits)
- Internship (SE502 - 2 TC)
- Project (SE503 - 2 TC)
- Capstone/Thesis (SE505/506/507 - 10 TC + 2 TC seminar)

## Teaching Plan (Highlights)
- **Semester 1**: Calculus, Linear Algebra, English 1, Intro to SE, Intro to Programming, OS.
- **Semester 2**: OOP, DSA, Discrete Structures, Stats, English 2.
- **Semester 3**: DB, Networking, Computer Architecture 2, Visual Programming, English 3.
- **Semester 4**: Intro to Software Engineering, Politics, Electives.
